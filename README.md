## 📌 Как получить API-ключ

1. Перейдите на [https://app.exchangerate-api.com/sign-up](https://app.exchangerate-api.com/sign-up)
2. Зарегистрируйтесь (бесплатный тариф — 1500 запросов/месяц)
3. После входа скопируйте ваш ключ из панели управления
4. Вставьте его в код в переменную `API_KEY`

## 🚀 Запуск

```bash
python currency_converter.py
